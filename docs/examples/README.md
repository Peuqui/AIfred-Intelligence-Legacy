# AIfred Intelligence - Example Showcases

This folder contains example conversations demonstrating AIfred's capabilities.

## Multi-Agent Tribunal: "Are cats better than dogs?"

A simple question turns into a philosophical debate between a British butler, an ancient Greek philosopher, and King Solomon.

**[View the full interactive debate](https://peuqui.github.io/AIfred-Intelligence/examples/cats-vs-dogs-tribunal.html)**

### What happens in this debate:

1. **AIfred (Butler)** provides a polite, balanced comparison table
2. **Sokrates (Critic)** points out that facts without prioritization don't answer the question
3. **Salomo (Judge)** synthesizes both views and requests a decision framework
4. **AIfred refines** his answer with a concrete decision framework
5. **Sokrates persists** - demands weighted scoring and philosophical clarity
6. **Salomo concludes** - practical solution achieved, consensus reached

### Highlights:

- AIfred maintains his Butler persona throughout ("Indeed", "I dare say", "most certainly")
- Sokrates is delightfully pedantic - demanding a numerical scoring system for pet preference
- Salomo uses Hebrew ("b'seder" = good/okay) for authenticity
- The debate escalates from a simple table to a weighted decision matrix
- Total runtime: ~8 minutes across 3 different LLMs
