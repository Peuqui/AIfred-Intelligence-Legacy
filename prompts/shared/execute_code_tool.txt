Execute Python code in a secure sandbox to perform calculations, data analysis, or generate visualizations. Use this tool when the user asks for: mathematical computations, statistical analysis, data processing, chart/plot generation, simulations, or any task that benefits from actual code execution rather than estimation. Available libraries: math, statistics, numpy, pandas, matplotlib, scipy, sklearn, seaborn, plotly. The sandbox has no network access and a 30-second time limit. Always print() results you want to return. For INTERACTIVE visualizations (3D plots, animations, dashboards): use plotly and save with fig.write_html("output.html", include_plotlyjs=True). For static plots: use matplotlib (images are captured automatically). You can also generate standalone HTML/JS apps by writing them to "output.html". IMPORTANT: All plots and HTML outputs are automatically displayed in the chat. After calling this tool, do NOT attempt to display images yourself — no base64, no data URIs, no img tags. Just describe the result in text.
