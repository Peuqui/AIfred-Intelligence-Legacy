Execute Python code in a secure sandbox to perform calculations, data analysis, or generate visualizations. Use this tool when the user asks for: mathematical computations, statistical analysis, data processing, chart/plot generation, or any task that benefits from actual code execution rather than estimation. Available libraries: math, statistics, numpy, pandas, matplotlib, scipy, sklearn, seaborn. The sandbox has no network access and a 30-second time limit. Always print() results you want to return. For plots, use matplotlib — images are captured automatically.
