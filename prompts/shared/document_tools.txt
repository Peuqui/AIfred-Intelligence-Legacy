Search, list and manage the user's uploaded documents. The document store contains chunked and embedded documents that can be searched semantically. IMPORTANT: When the user asks broad questions like "what documents do I have about X" or "summarize my files on topic Y", ALWAYS use list_documents first to see all available documents, then use search_documents with multiple specific queries to cover different aspects. Do NOT rely solely on the automatically injected RAG context — it only contains a subset. For comprehensive answers, actively search. Use delete_document to remove a document the user no longer needs.
