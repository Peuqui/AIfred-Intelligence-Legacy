# FORMAT A: MULTI-AGENTEN-TRACKING
═══════════════════════════════════════════════════════════════════════════

Pro Runde die TATSÄCHLICH stattfand (R1, R2, etc.):

```
R[N]:
AIfred: [Kernaussage in 1 Satz]
Sokrates: [Kritik in 1-2 Sätzen]
Salomo: [Empfehlung] → [LGTM/WEITER]
```

Am Ende NUR wenn letzte Runde erreicht:
```
Status:
  1. ✅/⚠️/❌ [Kritikpunkt]: [Status]
```

**KRITISCHE REGELN:**
- NUR Runden dokumentieren die TATSÄCHLICH in der Konversation vorkommen!
- KEINE Runden erfinden! Wenn nur R1 und R2 da sind, KEIN R3 schreiben!
- KEIN Markdown (keine ##, **, etc.) - nur Plain Text!
- Maximal {max_tokens} Tokens
- Kompakt: 1-2 Sätze pro Agent pro Runde

# FORMAT B: STANDARD-KOMPRESSION
═══════════════════════════════════════════════════════════════════════════

Erstelle eine **kompakte Liste** der Hauptpunkte:

**ANFORDERUNGEN:**
- Maximal {max_tokens} Tokens (ca. {max_words} Wörter)
- Nur Fakten, keine Meinungen
- Chronologische Reihenfolge
- Keine Wiederholungen
- Keine Höflichkeitsfloskeln
- Keine Meta-Informationen

**STIL:**
- Verwandte Fragen in einer Zeile kombinieren
- Themen mit Semikolon trennen
- Konkrete Zahlen/Werte direkt einbauen
- Konversationelles Hin-und-Her weglassen

**STRUKTUR:**
```
[Thema1]: [Kernfakten, Schlussfolgerungen]; [Thema2]: [Kernfakten, Schlussfolgerungen]; ...
```

**WICHTIG:** Kopiere diese Struktur NICHT wörtlich - passe sie an den tatsächlichen Konversationsinhalt an!
