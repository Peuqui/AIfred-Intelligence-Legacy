Extrahiere alle Daten aus diesem Bild als JSON.

Format:
```json
{
  "type": "table|list|form|text",
  "columns": ["Spalte1", "Spalte2"],
  "rows": [["Wert1", "Wert2"]]
}
```

Typen:
- table: {"columns": [...], "rows": [[...]]}
- list: {"items": [...]}
- form: {"fields": [{"label": "...", "value": "..."}]}
- text: {"content": "..."}
