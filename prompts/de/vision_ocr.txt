Extrahiere ALLE sichtbaren Texte, Daten und Strukturen aus den Bildern als JSON. Dies ist eine vollständige OCR-Aufgabe.

KRITISCH - MEHRERE BILDER:
- Wenn MEHRERE Bilder vorhanden sind, analysiere JEDES Bild EINZELN und vollständig!
- Jedes Bild bekommt einen eigenen Abschnitt in der JSON-Ausgabe
- Überspringe KEIN Bild - ALLE Bilder müssen analysiert werden!

KRITISCHE REGELN:
- Extrahiere NUR was TATSÄCHLICH in den Bildern steht - KEINE Ergänzungen, KEINE Korrekturen, KEINE Interpretationen!
- ERFINDE KEINE Daten oder Zeilen - wenn etwas unleserlich ist, lasse es leer oder schreibe "unleserlich"
- Kopiere Text EXAKT wie er dasteht - keine Rechtschreibkorrekturen, keine Verbesserungen
- Bei Tabellen: Zähle die Zeilen im Bild und extrahiere GENAU diese Anzahl - nicht mehr, nicht weniger!

WICHTIG:
- Erfasse ALLE Abschnitte: Header, Tabellen, Listen, Formulare, Fließtext
- Extrahiere JEDE Zeile und JEDE Spalte vollständig
- Bei mehreren Tabellen: Erstelle ein Array mit allen Tabellen
- Bei gemischten Inhalten: Kombiniere alle Strukturen in einer logischen Reihenfolge
- Lasse NICHTS aus - vollständige Rekonstruktion des Dokuments!
- Bei MEHREREN BILDERN: Jedes Bild als separater Eintrag in "images" Array!

Format für einzelne Struktur:
```json
{
  "type": "table|list|form|text",
  "columns": ["Spalte1", "Spalte2"],
  "rows": [["Wert1", "Wert2"]]
}
```

Format für mehrere Strukturen (ein Bild):
```json
{
  "type": "document",
  "sections": [
    {"type": "text", "content": "Header-Text"},
    {"type": "table", "columns": [...], "rows": [...]},
    {"type": "form", "fields": [...]}
  ]
}
```

Format für MEHRERE BILDER (WICHTIG!):
```json
{
  "type": "multi_image",
  "images": [
    {
      "image_index": 1,
      "description": "Kurze Beschreibung was im Bild zu sehen ist",
      "content": {"type": "document", "sections": [...]}
    },
    {
      "image_index": 2,
      "description": "Kurze Beschreibung was im zweiten Bild zu sehen ist",
      "content": {"type": "table", "columns": [...], "rows": [...]}
    }
  ]
}
```

Typen:
- table: {"columns": [...], "rows": [[...]]} - Tabellen
- list: {"items": [...]} - Listen
- form: {"fields": [{"label": "...", "value": "..."}]} - Formulare
- text: {"content": "..."} - Fließtext
- document: {"sections": [...]} - Mehrere Abschnitte

Antworte NUR mit gültigem JSON in einem Markdown-Code-Block!
